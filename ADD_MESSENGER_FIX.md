# Adding Messenger - Redirecting Back Means It's Not Added Yet

## Problem: URL redirects back to dashboard

This means Messenger **hasn't been added as a product yet**. You need to add it first!

---

## Method 1: Try Products Page with Setup

Try this URL (replace `YOUR_APP_ID` with your App ID):

```
https://developers.facebook.com/apps/YOUR_APP_ID/products/add/
```

This should show you available products to add, including Messenger.

---

## Method 2: Via "Use Cases" Section

Since you added "Facebook Login" as a use case earlier:

1. **In the left sidebar**, click on **"Use cases"**
2. **Look for:**
   - "Add use case" button
   - "Business messaging" or "Messenger" option
   - Any option related to messaging/chat

3. **Try to add "Business messaging" use case** - this might add Messenger automatically

---

## Method 3: Check Dashboard Content Carefully

1. **In the main dashboard area (center)**, look for:
   - Scroll ALL THE WAY UP to the top
   - Scroll ALL THE WAY DOWN to the bottom
   - Look for ANY blue or green buttons
   - Look for text like:
     - "Get started"
     - "Add integration"
     - "Set up"
     - "Connect"

2. **Try clicking** on any button that looks relevant
